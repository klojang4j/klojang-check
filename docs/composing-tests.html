KLOJANG CHECK

Composing Tests

Klojang Check allows you to combine two or more checks into a single new check that expresses the logical conjunction (AND) or disjunction (OR) of the individual checks. Generally that latter will be of more use since the chain of checks following Check.that(...) already constitutes a logical conjunction. For example, this statement:

    this.numChairs = Check.that(numChairs).is(positive()).is(lte(), 4).is(even()).ok();

requires the number of chairs to be positive and less than, or equal to 4 and even. If the number of chairs needs to pass just one of these tests, write:

    this.numChairs = Check.that(numChairs).is(positive().orElse(lte(), 4).orElse(even());

Nevertheless, you might still want to use the and() methods for conciseness:

    Check.that(text).is(notNull().and(hasSubstring(), allOf(), "to", "be", "or", "not");

which is equivalent to, but quite a bit more economical than:

    Check.that(text).is(notNull())
        .is(hasSubstring(), "to")
        .is(hasSubstring(), "be")
        .is(hasSubstring(), "or")
        .is(hasSubstring(), "not");
