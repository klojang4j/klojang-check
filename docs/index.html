Defensive Programming with Klojang Check

Klojang Check is a Java module dedicated to defensive programming. It provides a set of syntactical constructs that make it easy to express checks on program input, object state, method arguments, variables, computational outcomes and program output. In addition, it comes with some fifty predefined common checks on these program elements. These checks are associated with short, informative error messages, so you don't have to invent them yourselves.

To use Klojang Check, add the following dependency to your POM file:

<dependency>
  <groupId>org.klojang.check</groupId>
  <artifactId>klojang-check</artifactId>
  <version>3.0.2</version>
</dependency>

The Javadocs for Klojang Check can be found here.
